# parabola-forms

Reference module showing the standard forms of a parabola in two layout modes (table and list) and two vertex modes (origin and translated). Includes a `ParabolaIllustration` component that renders an accurate SVG diagram for any selected direction. Deployed to `/vertex/modules/parabola/forms/`.

## Controls

- Vertex mode toggle: origin `(0, 0)` or general `(h, k)`
- Layout toggle: compact table view or expanded list view with inline illustration
- In list view, tapping a row opens a detail modal with a full illustration
