# parabola-cards

Swipeable reference card deck covering the four standard forms of a parabola. Each card shows the equation, focus, directrix, axis of symmetry, and a small orientation diagram. Deployed to `/vertex/modules/parabola/cards/`.

## Interaction

- Arrow buttons and dot indicators for mouse/trackpad users
- Left/right swipe for touch
- Arrow key navigation for keyboard users
- Entry animations distinguish forward from backward traversal

## Dev

```bash
pnpm dev
```

Runs on port `5177`.
